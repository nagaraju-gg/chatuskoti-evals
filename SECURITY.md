# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.3.x   | ✅ Active |
| 1.2.x   | ⚠️ Critical fixes only |
| 1.1.x   | ❌ Unsupported |

## Reporting a Vulnerability

This is a research demonstration project. If you discover a security vulnerability,
please report it by emailing **nagaraju.raj95@gmail.com**.

Please do not open public issues for security vulnerabilities.

We will acknowledge receipt within **48 hours** and provide an estimated timeline
for a fix. Security fixes will be backported to supported versions as patch releases.
