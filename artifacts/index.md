# Artifact Index

This directory collects torch-backed evidence bundles for `Chatuskoti Evals`.

Generated by `scripts/run_torch_bundle.sh` for the current package version.

## How to regenerate

```bash
# Run the bundle generator (defaults to artifacts/strong_v{version}_torch)
./scripts/run_torch_bundle.sh

# Or specify a custom output directory
./scripts/run_torch_bundle.sh artifacts/custom_bundle 15 3 300
```

The bundle script (`scripts/run_torch_bundle.sh`) automatically stages the latest artifacts in git and replaces any previously tracked `strong_v*` bundle. Old versions live in git history, not on disk — run the script again to regenerate.
